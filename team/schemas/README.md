# Team Schemas

Team-specific schemas belong here.

Product-neutral research schemas live in:

```text
shared/research/schemas/
```

The existing `paper-card.schema.json` and `topic-profile.schema.json` files are compatibility artifacts from the first Team scaffold. New implementation should use the shared schemas first, then add Team-specific schemas for review state, permissions, project libraries, assignments, briefs, and audit events.
